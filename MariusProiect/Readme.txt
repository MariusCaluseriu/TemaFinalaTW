Tehnologii folosite in proiect:

->HTML (HyperText Markup Language) este limbajul de marcare utilizat pentru a crea pagini web și pentru a structura conținutul acestora. Este utilizat pentru a defini structura și semantica elementelor de pe o pagină web, cum ar fi titlul, paragrafele, imaginile, link-urile etc.

->CSS (Cascading Style Sheets) este un limbaj de stilizare utilizat pentru a defini aspectul și designul paginilor web create cu HTML. CSS este utilizat pentru a seta proprietățile de stilizare, cum ar fi culoarea, dimensiunea, poziționarea și fonturile, astfel încât pagina web să arate așa cum doriți.

->JavaScript este un limbaj de programare utilizat pentru a crea interactivitate pe paginile web și pentru a controla comportamentul acestora. JavaScript este utilizat pentru a crea funcții de validare a formularelor, pentru a crea animații și efecte vizuale, pentru a manipula elementele de pe pagină și pentru a crea aplicații web complexe. Este un limbaj de programare foarte popular și esențial pentru dezvoltarea de pagini web moderne.
